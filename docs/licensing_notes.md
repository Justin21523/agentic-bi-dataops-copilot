# Licensing Notes

This project is designed around derived features and metadata. Public demos must not include complete lyrics, raw lyric text, or line-by-line lyric text.

Allowed display surfaces include BoW terms, topic labels, sentiment scores, style vectors, metadata, aggregations, and similarity results.

CSV exports are restricted to metadata, derived features, model outputs, and aggregates. They must not include full lyric text, lyric lines, or raw lyric fields.
