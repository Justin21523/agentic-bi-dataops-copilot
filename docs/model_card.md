# Model Card

MVP baselines:

- TF-IDF features from BoW terms
- NMF topic modeling
- proxy sentiment scoring from weighted terms
- logistic regression genre classifier
- cosine similarity over TF-IDF vectors
- topic-vector and style-embedding similarity variants
- deterministic style vectors for artist fingerprints

The dashboard also exposes derived cultural summaries for decade term evolution, genre sentiment, language/culture distribution, artist style clusters, and genre leakage in similarity results.

These models are intended for demonstration and integration testing.
